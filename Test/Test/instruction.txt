Управление клавишами:
// все действия происходят над фокусирующими фигурами
"Стрелки" 					 - перемещение фигуры
Q/E                          - поворот фигуры влево/вправо
W/S                          - увеличение/уменьшение масшатаба по всем осям
X                            - вернуть предыдущее состояние фигуры
Z                            - вернуться первичное состояние фигуры
C                            - переключить режим "следа" фигуры
V                            - переключить режим "видимости" фигуры
Page Up/Down                 - сменить фокус на следующую/предыдущую фигуру
// только на "нампаде"
+                            - создать фигуру
-                            - удалить фигуру

Продвинутые команды:
// 0   - обязательное число
// 1-9   - необязательное число
// ... - бесконечный список параметров
// нумерация фигур начинается с 0

CreateCircle_1_2_3_4_5_6_7_8 - создаёт круг в точке (2, 3) с радиусом 4 и цветом (5, 6, 7, 8)
CreateAgregat                - создаёт агрегат из фокусирующих фигур
	CreateAgregat_...        - создаёт агрегат из выбранных по номеру фигур

ChangePosition_0_1   	 	 - меняет позицию
ChangeAngle_1        	 	 - меняет угол
ChangeScale_1_0     	 	 - меняет масштаб

SetPosition_0_1       	 	 - установить позицию
SetAngle_0            	 	 - установить угол
SetScale_0_1          	 	 - установить масштаб
	SetScaleX_0       	 	 - установить масштаб по оси абцис
	SetScaleY_0       	 	 - установить масштаб по оси ординат
SetColor_0_1_1_1      	 	 - установить цвет RGBA
SetFocus_...          	 	 - установить фокус на фигуры

SetSpeed_0            	 	 - установить скорость выполнения программы   // системная настройка, изначально равна 800
SetMove_0             	 	 - установить скорость передвижения фигур     // ситсемная настройка, изначально равна 1
SetRotate_0           	 	 - установить резкость поворота фигур         // системная настройка, изначально равна 10
SetScale_0            	 	 - установить резкость масштабируемости фигур // системная настройка, изначально равна 1

Delete                	 	 - удалит фокусируемые фигуры
	Delete_...        	 	 - удалит выбранные по номеру фигуры
	DeleteAll         	 	 - удалит все фигуры

SwitchTrace           	 	 - переключит режим "следа" у фокусирующих фигур
	SwitchTrace_...   	 	 - переключит режим "следа" у выбранных по номеру фигур
	SwitchTraceAll    	 	 - переключит режим "следа" у всех фигур
SwitchVisible         	 	 - переключит режим "видимости" у фокусирующих фигур
	SwitchVisible_... 	 	 - переключит режим "видимости" у выбранных по номеру фигур
	SwitchVisibleAll  	 	 - переключит режим "следа" у всех фигуры
SwitchFocusUp         	 	 - сменит фокус на следующую фигуру
SwitchFocusDown       	 	 - сменит фокус на предыдущую фигуру

Wipe                  	 	 - вернёт предыдущее состояние фокусирующих фигур
WipeAll               	 	 - вернёт первичное сочтояние фокусирующих фигур
